Standard Responses

Standard (or canned) responses can be a huge timesaver if you are dealing with large volumes of tickets. This enables you to select a pre-formatted message, rather than keep repeating yourself with ticket responses.

Standard responses can be grouped into categories for convenience and contain HTML if required. To access the standard response feature select 'Tickets>Standard Responses' from the admin menu.

Title

Enter title for standard response. This is to easily identify this response.

Department

Specify which departments this response applies to. Can be ALL departments or specific department. When viewing tickets, reponses only applicable to all or individual department will be shown.

Answer

Enter standard response answer. For formatting options, use the available BBCode if enabled. HTML is not allowed.

Enable/Disable

Do you wish to enable or disable this response? You can also batch enable/disable responses.

Preview, Edit & Delete

Use the links provoded to update, preview or delete responses.

Usage

When answering a ticket or dispute, click the 'Standard Responses' link above the textarea to display drop down of available responses.

Help Tips

The same or additional information may be provided by hovering your cursor over the image next to each heading in the admin area if applicable.
