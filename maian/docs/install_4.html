Language

ALL files related to the default language are found in the following directory:

templates/language/english/*

To create a new language pack, make a copy of the 'english' folder, translate the files and then enable new language in settings. Keep language folder name to lowercase. ie: french, german, finnish etc

General Language

ALL general language is found in the PHP files in the above folder:

templates/language/english/lang1.php
templates/language/english/lang2.php

New files may be added with new releases. Use a good text editor to edit. Notepad should be fine. DO NOT edit in a word processor.

E-Mail Template Language

All the language in the e-mails is in the e-mail template txt files:

templates/language/english/email/*.txt
templates/language/english/admin-email/*.txt

Again, use a good text editor to edit.

Multi Language Interface

It is possible to run multiple languages from the same install. Firstly, you need to enable this via an option in the 'control/user-defined/defined2.inc.php' file:

define('LANG_SWITCH_ENABLE', 1);
define('LANG_SWITCH_COOKIE', 180);

You should also set a cookie expiration time in days.

Next you need to make copies of the 'templates/language/english/' folder. Once copied, translate files. Lets say you now have:

templates/language/english/
templates/language/french/
templates/language/german/

Each language is loaded via a 'l' (Lowercase 'L') parameter in the url. If the language doesn`t exist, it will default to your language default setting.

So, lets say you have the above 3 languages, 'english','french' & 'german'. Links would be:

index.php?l=english (For English)
index.php?l=french (For French)
index.php?l=german (For German)

Where the 'l' value is the folder name for your language in the 'templates/language/' folder. When visitor uses an alternative language, all replies sent from admin will use the corresponding language email templates. Default load language will be the language set in your settings.

Language folders MUST contain only letters, numbers and hyphens. If you disable the switch, any cookies set on user machines will auto delete on the next visit.
