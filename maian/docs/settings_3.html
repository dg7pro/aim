Overview

This feature enables you to add custom form fields to the ticket creation and/or reply pages. Useful if you need to get specific data from customers or visitors on ticket creation or replies. Fields can be on ticket creation page only, as well as visitor replies and admin only reply pages.

Instructions/Text

Enter field question, text or instructions for customer/visitor. Max 250 Chars.

Field Type

Which field type do you require?

Textarea = Multi line
Input = Single line (Max 250 Chars)
Select - Single selection from multiple options
Checkboxes = Multi selections from multiple options.

Required Field

Is this a required field? If yes, customer/visitor must complete option. If checkboxes are used, at least one box must be selected. Note that restrictions are not imposed in admin area.

Select/Checkbox (One per line)

For select and checkboxes which require multiple options. ONE entry PER line.

Apply to Department(s)

Specify which ticket departments the custom field will be available in when the department is selected on the create ticket page. If none are specified it will be shown for ALL departments.

Field Location

Where should the new fields appear?

New Ticket = Added to ticket creation page
Ticket Reply = Added to ticket reply page
Admin Reply = Added to admin ticket reply page

If none are selected, will default to New Ticket.

Show For All Replies

If a field location is set as an Admin Reply it appears on the admin reply pages only. Do you want visitors to see this field information data on their ticket pages? If no, seen only by admin. Useful for backend only operations.

Enable Field

Do you wish to enable this field? You can also batch enable/disable fields via the checkboxes.

Field Ordering

You can specify the order of the fields appearance via the drop down menus provided.

Edit/Delete Options

Use the edit/delete options to update or delete fields.

Help Tips

The same or additional information may be provided by hovering your cursor over the image next to each heading in the admin area if applicable.
