Overview

Here you can add some frequently asked questions into the system and group them into the various categories. Users can see the F.A.Q on the main ticket page and see if their question has already been answered.

Enable the F.A.Q in the settings. You can batch add F.A.Q entries for quickness. Use BB Code for formatting if enabled.

Question

Enter F.A.Q question. Use BBCode formatting if enabled to enhance question text.

Category

Specify which category this question applies to. Can be ALL categories or specific category.

Status

Enable or disable question.

Attachments

This appears if there are 1 or more F.A.Q attachments. Click the link to display attachments and click checkboxes to assign attachments to question.



Answer/Preview

Enter F.A.Q answer. For formatting options, use the available BB code if enabled. HTML is not allowed. Click 'Preview Answer' to preview how text will look to visitor.

Reset Voting Counts

Use the checkboxes to reset voting counts for certain questions.

Preview/Edit/Delete

Click the links provided to preview existing questions or to edit or delete.

Help Tips

The same or additional information may be provided by hovering your cursor over the image next to each heading in the admin area if applicable.
