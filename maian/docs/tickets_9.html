Overview

This feature lets you assign tickets to specific users. Only users assigned to specific tickets can see them, along with the administrator who by default, can always see all tickets. Use the checkboxes provided to assign tickets. Tickets can be assigned to a single user or multiple users. If you wish to assign a ticket later, leave all checkboxes blank.

Filters

Use the filters provided to view specific tickets.

Assign To

Use the checkboxes to select users to assign ticket to. Only users who have email notification enabled are shown.

Note that further notifications are only sent to assigned users, so if you are the administrator and want to receive ticket reply notifications, you should always add yourself to the assigned list.

View Full Ticket

Click this to view the full ticket. Here you can edit the ticket data before assigning it to a user.

E-mail notification to users

Check box to send email to users, this notifies the user that at least 1 ticket is awaiting a reply. Note that notification is NOT sent to the user logged in executing the assign.
