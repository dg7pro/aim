Main Features

Easy to install support ticket system written in PHP/MySQL
Multiple users with access levels
Multiple departments
Fully searchable F.A.Q with categories, sub categories, attachments and ajax based voting system
Validates as XHTML Strict. Virtually tableless design
Template based for easy site integration
E-mail templates for easy editing
SMTP mail options
Option to manually assign tickets to users by admin
Supports BBCode for text formatting and video display via YouTube/Vimeo
Optional spam prevention via reCaptcha
Ability to start tickets via e-mails (uses IMAP functions)
Multiple attachments with file/size restrictions
Ticket portal to view tickets by e-mail address
Create disputes from tickets so multiple people can respond to the same ticket
Search tickets in portal
Option for visitors to change passwords
Standard (or canned) responses to speed up response times
Batch import standard responses and knowledge base entries
Timezone support
Ability for admin to assign tickets to users/staff
Options for admin to reset passwords and transfer tickets between e-mail accounts
Option for admin to merge tickets
Message option to easily contact other support team members
Search function to locate tickets
Add support team user signatures to outgoing emails
Add custom fields. Available per department
Basic reports screen for overview of ticket totals
Noteboard option for admin to make backend notes about tickets
Open tickets via XML-RPC data post API. Any application/device that can trigger XML posts can open tickets
Option to auto close tickets that have had not been replied to. Optional email notification
Admin entry log to monitor log in activity
View user replies
Graphical statistics for tickets/replies
User graphical statistics for replies
