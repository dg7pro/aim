Template Integration

Maian Support utilises the Savant2/3 template engines. These make integrating the layout into your existing site relatively easy compared with none templated systems. The templates for the system are located in the templates folder. Other HTML has been separated from the PHP code and can be found in the templates/html/ folder.

Where you see PHP variables, these should NOT be removed. Example:

<?php echo $this->CODE; ?>

See comments in templates for certain variables. The .htm template files have variable braces. ie: {brace}. These should NOT be removed either.

To integrate Maian Support into your existing site design is going to require some basic knowledge of HTML. You`ll start by looking at your current HTML structure and comparing it to that of Maian Support. Generally work with the header and footer first. If you don`t understand HTML, integrating into another site layout is NOT recommended. It is also NOT recommended you integrate the layout using PHP includes or iframes.

The main 'index.php' files and files in the 'core' folders are encoded and cannot be changed.

Parse Custom PHP Code from Main Index Files

If you want to load custom code to parse from the main index.php files, add your code to the following files:

control/index-parser.php
admin/control/index-parser.php

Add Your Own Logos

If you wish to add your own logos, simply overwrite the default images:

templates/images/logo.png
admin/templates/images/logo.png

E-Mail Templates

Besides HTML templates, Maian Support also makes it easy to modify e-mail data. The e-mail templates are located at:

templates/language/english/email/*.txt
templates/language/english/admin-email/*.txt (admin e-mail templates can be edited)

You can open these templates and add whatever text you want. A good quality text editor is recommended. DO NOT use word processors. Notepad is fine. You should be very careful NOT to remove any code between braces. ie: {name} Removing these variables will cause some data not to be included in e-mails.

HTML in E-Mail Templates

If you want to use HTML in emails, enable the option in the "control/user-defined/defined.inc.php" file:

define('HTML_EMAILS', 1);

If you enable this you need to change the email .txt files and use standard html code. All carriage returns must be replaced with html line breaks etc

Colours & Appearance

For many people, changing the colours and a few default images is good enough. Maian Support utilises Cascading Stylesheets for colour display. ALL of the colours you see for the default design are in the stylesheets. Maian Support has the following stylesheets, all of which can be edited in a good text editor.

stylesheet.css (Frontend)
greybox.css (Frontend - Popup Window Styling Only)
ie.css (Internet Explorer Hacks)

admin/stylesheet.css (Admin)
admin/jquery-ui.css (Admin - JQuery UI Styles. etc Date Picker)
admin/menu.css (Admin - Menu)
admin/greybox.css (Admin - Popup Window Styling Only)

Edit these and choose whichever colours you want.

Images

Images are located in the following folders:

templates/images/ (Frontend)
admin/templates/images/ (Admin)
